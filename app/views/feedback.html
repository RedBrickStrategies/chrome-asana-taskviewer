What is this?

This is a non-offical chrome app for Asana using their API. The core aim of this app is to allow the user fast and easy access to their tasks, and allow them to update its status, comment or attach files straight from desktop. It also syns stuff offline, so you can view your tasks with or without an internet connection. Finally it also as an auto-refresh built in, notifying you of changed tasks in the project / workspace you are currently active.
